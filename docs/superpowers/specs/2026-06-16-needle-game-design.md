# 见缝插针游戏设计规格

## 概述

为游戏控制台新增见缝插针小游戏（`needle`）。旋转圆盘上插着针，玩家在合适时机发射新针，插入空隙得分，碰到已有针则失败。

## 玩法

### 状态机
- `needle_state_ready` — 圆盘旋转中，等待发射
- `needle_state_flying` — 针飞行中
- `needle_state_over` — 碰撞失败

### 流程
1. **Ready**：圆盘旋转，底部有待发射针，按按钮发射
2. **Flying**：针沿直线飞向圆心，到达后插入，新针出现在底部
3. **Over**：碰撞 → 显示 GAME OVER → 排行榜

### 碰撞检测
- 发射针接近圆心时（距离 < 针长度），计算其相对圆心的角度
- 若该角度与任何已插入针的角度差 < 阈值（约 12°）→ 碰撞失败
- 否则插入成功，该角度加入已有针集合

### 难度递增
- 每插入 5 根针，圆盘角速度 +1
- 初始角速度：2°/帧（~100°/s at 50fps）

### 操作
| 输入 | 行为 |
|------|------|
| 按钮按下 | 发射针 |
| 摇杆左右 | 待发射针水平摆动 |
| 长按 1s | 返回菜单 |
| 任意方向（结束） | 重来 |

## 视觉效果

### 绘制方式
- **圆盘**：3 层嵌套 Fill_Rect 正方形（50×50 深灰 / 40×40 中灰 / 30×30 浅灰），视觉近似圆
- **针尖**：4×4 小方块，颜色与插入顺序对应（七彩循环）
- **发射针**：4×12 细长矩形（2×12 更好），从底部指向圆心
- **增量绘制**：每帧擦旧针尖 → 更新角度 → 画新针尖

### 颜色
| 元素 | 颜色 |
|------|------|
| 圆盘外层 | `0x4208` |
| 圆盘中层 | `0x8410` |
| 圆盘内层 | `0xc618` |
| 针尖 | 赤橙黄绿青蓝紫循环 |
| 发射针 | `0xffff` |
| 背景 | `0x0000` |

## 计分

- 每插入一根针：+1
- 总分 = 插入针数
